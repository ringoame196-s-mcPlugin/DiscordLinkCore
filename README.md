## プラグインダウンロード

## コマンド
| コマンド名       |     説明      | 権限 |
|-------------| ----------- | ------- |
| コマンド名       | 説明 | 権限 |
| /dcore link | Discordアカウントと連携する | ----|
| /dcore unlink | Discordアカウントとの連携を解除する | ---- |
| /dcore show | 自分自身の連携情報を出力 | ---- |
| /dcore show <ターゲット> | ターゲットの連携情報を出力 | op |
